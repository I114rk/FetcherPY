# 🚀 FetcherPY

> Полностью завайбкоженый, современный и быстрый аналог `neofetch`, написанный на Python с использованием библиотеки `Rich`. Специально для тех, кто ценит TrueColor градиенты и эстетику в своём терминале.

---

## ✨ Особенности

* 🎨 **TrueColor Градиенты:** Никаких унылых 8-битных цветов — только сочные 16-миллионные Hex-градиенты.
* ⚡ **Быстрота:** Минималистичный сбор информации о системе с помощью `psutil`.
* 📦 **Поддержка пакетных менеджеров:** Из коробки считает `pacman` и `flatpak`.
* 🖥️ **Умный сбор GPU:** Точечный парсинг `lspci` без лишнего мусора в строке видеокарты.
* 🎛️ **Красивый TUI:** Раздельные панели для ASCII-арта и системных данных с аккуратными рамками.

## 🛠️ Системные требования

Для корректной работы утилиты вам понадобятся:
* **Python** >= 3.10
* Библиотеки: `pip install rich psutil`
* *Опционально:* `wmctrl` (для корректного определения WM в X11 сессиях)

---

## 🚀 Установка и запуск

### Клонирование репозитория
```bash
git clone [https://github.com/i114rk/FetcherPY.git](https://github.com/i114rk/FetcherPY.git)
cd fetcherpy
```
##Настройка окружения (рекомендуется)


```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
(Не забудь создать файл requirements.txt с текстом rich и psutil внутри)
##Запуск
```bash
python fetcherpy.py
```
